# Simple Pico y placa predictor ⛏ 🚗

## Instalation
This project was developed using Node v16, npm, TypeScript and jest

```bash
npm install
```

## How tos

### Run the project

```bash
npm run start
```

### Run in dev mode
```bash
npm run start:dev
```

### Run test
```bash
npm run test
```
or 
```bash
npm t
```

To run the test in watch mode use: 
```bash
npm run test:watch
```
